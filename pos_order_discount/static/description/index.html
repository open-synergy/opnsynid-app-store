Basic Features

  • POS global discount allows you to provide flat discount on total invoice.
  • There can be multiple discounts created on different basis.
  • Only single discount is applicable per invoice.
  • Under POS configurations, select discounts to be applicable.
  • Discount application can be tax inclusive or exclusive.
